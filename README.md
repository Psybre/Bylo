# Bylo — La IA que agenda tus citas

Demo interactiva de Bylo, una plataforma de gestión de citas con IA, CRM y fidelización de clientes.

## Demo en vivo

**[bylo-ivory.vercel.app](https://bylo-ivory.vercel.app)**

## Tecnologías

- React 18 (via CDN)
- Babel Standalone (JSX en el navegador)
- CSS personalizado sin dependencias externas
